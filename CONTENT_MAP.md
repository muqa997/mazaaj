# خريطة محتوى الموقع — أين أعدّل كل نص؟

هذا الملف يدلّك على **المكان بالضبط** لتعديل أي نص أو كتابة موجودة بالموقع. كل قسم أدناه يذكر: (1) اسم الملف، (2) "المفتاح" (key) اللي تدوّر عليه داخل الملف.

## كيف تستخدم هذا الملف

1. افتح مشروع الموقع بمحرر الأكواد (VS Code).
2. افتح الملف المذكور من قائمة الملفات على اليسار (أو اضغط `Ctrl+P` واكتب اسم الملف للوصول له بسرعة).
3. اضغط `Ctrl+F` وابحث عن "المفتاح" المذكور (مثلاً `"titleLine1"`) لتوصل للسطر مباشرة.
4. عدّل النص الموجود بين علامتي التنصيص `" "` فقط، **لا تلمس اسم المفتاح نفسه ولا علامات `{ } , :`**.
5. احفظ الملف (`Ctrl+S`).

## قاعدة مهمة: الموقع بلغتين

كل نص بالموقع موجود بملفين متطابقين بالبنية:
- `messages/ar.json` → النسخة العربية
- `messages/en.json` → النسخة الإنجليزية

**إذا عدّلت نص بالعربي، لازم تعدّل مقابله بالإنجليزي بنفس الملف الآخر بنفس المفتاح**، وإلا تظهر صفحة الموقع بالإنجليزي بنص قديم لا يطابق العربي.

## 3. المنيو — نوعان منفصلان

### أ) المنيو التفاعلي للعرض (`/menu`) وb) منيو الطلب المرتبط بالسلة (`/order`)
كلاهما يعرضان **نفس البيانات** من ملف واحد:

| القسم | الملف |
|---|---|
| كل الأصناف، الأسعار، الأقسام (14 قسم)، وتسعير الأراجيل الثلاثي | `lib/menu-data.ts` |

**لإضافة/حذف/تعديل أي صنف أو سعر:** افتح `lib/menu-data.ts` مباشرة وعدّل السطر الخاص بالصنف — كل صنف على هذا الشكل:
```
{ id: "cc1", category: "coffeeClassic", name: { ar: "اسبريسو سنكل", en: "Espresso Single" }, price: 2500 },
```
غيّر النص بعد `ar:` للعربي، بعد `en:` للإنجليزي، وبعد `price:` للسعر (بدون فواصل أو رمز عملة).

**لإضافة قسم جديد كلياً** (مثل "حلويات" الذي أضفناه): يحتاج تعديل 3 أماكن، أخبرني وأنا أضيفه لك مباشرة — أسهل من التعديل اليدوي.

| اسماء الأقسام (التابات) بالعربي/الإنجليزي | الملف | المفتاح |
|---|---|---|
| كل اسم قسم (قهوة سادة، حلويات، أراجيل...) | `messages/ar.json` | `menuPage.categories.*` **و** `interactiveMenu.categories.*` (لازم تعدّل الاثنين معاً لأنهما مستقلان) |
| تسميات قواعد الأراجيل (خشب/بابلي/طبيعي) | `messages/ar.json` | `shishaBase.*` |
| عنوان/وصف صفحة منيو الطلب | `messages/ar.json` | `menuPage.title`, `menuPage.subtitle` |
| عنوان/وصف المنيو التفاعلي للعرض | `messages/ar.json` | `interactiveMenu.title`, `interactiveMenu.subtitle` |
