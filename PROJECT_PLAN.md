# كافيه مزاج — خطة المشروع وتوثيقه

آخر تحديث: 2026-07-05

**مستودع GitHub:** https://github.com/muqa997/mazaaj

## الفكرة العامة

موقع ويب فاخر وتفاعلي (وليس تطبيق موبايل) لكافيه مزاج، يخدم عدة أهداف:

1. **عرض هوية الكافيه**: القصة، الأجواء، الطابق الثاني (صالة الألعاب).
2. **منيو تفاعلي بصفحة واحدة** (`/menu`): يُستخدم كوجهة لرابط QR الموضوع على طاولات الكافيه، ليتصفح الزبون المنتجات من جواله بدل المنيو الورقي.
3. **طلب توصيل** (قهوة، حلويات، أراجيل): بدون أي نظام حسابات مستخدمين — سلة بسيطة، ثم نموذج بيانات، ثم إرسال الطلب كرسالة جاهزة عبر واتساب ليأكده فريق الكافيه يدوياً.
4. **عرض خدمات الكافيه، موقعه على الخريطة، وروابط التواصل الاجتماعي**.
5. **وضع داكن (Dark Mode)** كامل، ودعم لغتين (عربي/إنجليزي).

الموقع **Mobile-First** (95% آيفون)، لكنه مصمم كموقع ويب حقيقي (Navbar + سكرول بين أقسام) وليس بواجهة تطبيق (لا يوجد Bottom Tab Bar).

## القرارات المعمارية المتفق عليها

| القرار | الخيار المُعتمد | السبب |
|---|---|---|
| هيكلة الموقع | صفحة رئيسية طويلة بالسكرول (أقسام) + صفحات مستقلة لـ `/menu` و `/order` | يحس الزائر إنه موقع تفاعلي حقيقي، ويسهّل ربط QR بصفحة منيو منفصلة |
| اللغتان | next-intl مع رابط لغة صريح (`/ar`, `/en`)، والعربية دائماً الافتراضية (بدون كشف تلقائي حسب المتصفح) | أفضل لمحركات البحث ومشاركة الروابط |
| نظام الطلبات | بدون تسجيل دخول/حسابات — سلة محلية (localStorage) + نموذج بيانات + تسليم الطلب كرسالة واتساب جاهزة | الكافيه ما يحتاج نظام باك-إند معقد أو بوابة دفع بهالمرحلة |
| الوضع الداكن | متغيرات CSS (`--color-background/primary/accent`) تتبدل مع صنف `.dark`، عبر مكتبة `next-themes` | يخلي كل الموقع ينقلب تلقائياً بدون تكرار كود بكل مكوّن |
| قسم "من نحن" | داخل الفوتر نفسه (مو قسم مستقل بمنتصف الصفحة) — لوجو دائري فوق فقرة تعريفية | طلب المستخدم إنه يكون بمكان "نبذة عنا" الموجود أصلاً بالفوتر |
| هيدر الجوال | الهمبركر والسلة يتبدلون جهتهم تلقائياً حسب اتجاه اللغة (RTL/LTR) عبر flexbox عادي بدون أي شرط برمجي، واللوجو دايماً بالنص | يطابق الاتجاه الطبيعي للغة بدون كود إضافي |

## هيكلة الملفات وشرح كل جزء

```
mazaaj/
├── middleware.ts              # يوجّه الزائر دائماً إلى /ar افتراضياً (أو /en إذا اختارها)
├── i18n/
│   ├── routing.ts             # تعريف اللغتين المدعومتين، الافتراضية (ar)، وتعطيل الكشف التلقائي
│   ├── navigation.ts          # Link/useRouter/usePathname بنسخة "واعية باللغة"
│   └── request.ts             # يحمّل ملف الترجمة المناسب حسب اللغة الحالية
├── messages/
│   ├── ar.json                # كل نصوص الموقع بالعربي
│   └── en.json                # نفس النصوص بالإنجليزي (لازم يبقوا متطابقين بالمفاتيح)
├── app/
│   ├── globals.css            # متغيرات ألوان الوضع الفاتح/الداكن، إعدادات آيفون، الخطوط، safe-area
│   ├── icon.png                # أيقونة الموقع (Favicon) — من public/logo.png
│   └── [locale]/
│       ├── layout.tsx         # الـ layout الجذري: الخطوط، ThemeProvider، NextIntlClientProvider،
│       │                        CartProvider، ويعرض Navbar + Footer + CartDrawer
│       ├── page.tsx            # الصفحة الرئيسية: تجمع كل الأقسام بالترتيب
│       ├── menu/page.tsx       # صفحة المنيو التفاعلية (وجهة رابط QR)
│       └── order/page.tsx      # صفحة إتمام الطلب (نموذج البيانات + إرسال واتساب)
├── components/
│   ├── Navbar.tsx              # شريطين منفصلين: شريط جوال (همبركر/لوجو وسط/سلة، يتبدل مع اللغة)
│   │                             وشريط ديسكتوب (لوجو، روابط تنقل، سلة)
│   ├── Footer.tsx              # فقرة "من نحن" (لوجو + نص) أعلاه، ثم أعمدة: روابط سريعة، تواصل، إعدادات
│   ├── Logo.tsx                # مكوّن اللوجو الموحّد (يستخدم mazaaj-logo-trimmed.png الشفاف)
│   ├── LanguageSwitcher.tsx    # زر بشكل كبسولة (سهم + اسم اللغة + أيقونة كرة أرضية)
│   ├── DarkModeToggle.tsx      # مفتاح تبديل الوضع الداكن/الفاتح (هلال/شمس بمؤشر منزلق)
│   ├── CartDrawer.tsx          # درج جانبي يعرض محتويات السلة ويقود لصفحة /order
│   ├── Hero.tsx                # قسم الافتتاح بالصفحة الرئيسية
│   └── sections/
│       ├── MenuPreview.tsx     # لمحة عن فئات المنيو + زر يوصل لصفحة /menu
│       ├── GameLounge.tsx      # قسم صالة الألعاب بالطابق الثاني (تصميم معكوس الألوان دائماً)
│       ├── DeliveryCTA.tsx     # قسم يشجع على طلب التوصيل
│       └── ContactSocial.tsx   # العنوان، أوقات الدوام، خريطة Google Maps، روابط التواصل الاجتماعي
├── lib/
│   ├── cart-context.tsx        # إدارة السلة بالكامل (إضافة/حذف/تعديل كمية) + حفظها بالمتصفح
│   ├── config.ts                # رقم واتساب/هاتف الكافيه الفعلي، روابط السوشال ميديا، إحداثيات الخريطة
│   ├── motion.ts                # حركة ظهور موحدة (fade-up) تُستخدم بكل الأقسام
│   └── placeholder-menu.ts      # ⚠️ منتجات تجريبية مؤقتة لحد ما توصلنا قائمة المنيو الفعلية
├── public/
│   ├── logo.png                 # اللوجو بخلفية بنية (مستخدم بالفوتر وأيقونة الموقع)
│   ├── mazaaj-logo.png          # اللوجو بخلفية شفافة (النسخة الأصلية من المستخدم)
│   └── mazaaj-logo-trimmed.png  # نفس اللوجو الشفاف لكن مقصوص بدون هوامش فارغة (يُستخدم بالهيدر)
└── tailwind.config.ts           # نظام الألوان (يقرأ من متغيرات CSS) والخطوط
```

## أشياء لسه ناقصة — أحتاجها منك

- ⚠️ **قائمة المنيو الحقيقية** (الفئات، الأصناف، الأسعار، وصور إذا متوفرة) — حالياً فيه بيانات تجريبية بملف [lib/placeholder-menu.ts](lib/placeholder-menu.ts). هذا آخر شي متبقي، وراح يصمم المنيو كصفحة تفاعلية منفصلة حسب طلبك.

كل باقي البيانات (رقم التواصل، العنوان، أوقات الدوام، حسابات التواصل الاجتماعي، اللوجو، موقع الخريطة) صارت فعلية ومحدّثة.

## ملاحظة بخصوص اللوجو والوضع الداكن

اللوجو الشفاف الحالي (`mazaaj-logo-trimmed.png`) لونه بني، فيبين بشكل ممتاز بالوضع الفاتح، لكن يصير أخف وضوحاً بالوضع الداكن (تباين أقل مع الخلفية الغامقة). إذا حبيت أزيد وضوحه بالوضع الداكن تحديداً (مثلاً بإضافة دائرة خلفية فاتحة خلفه بذاك الوضع بس)، خبرني وأسويها.
